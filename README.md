# elixir-web-phoenix-traefik-ssl-hello-world

## Description
A POC elixir phoenix project rendering html.
Uses self signed ssl certs.

## Tech stack
- ssl
- self signed

## Docker stack
- alpine:edge
- bitwalker/alpine-elixir-phoenix:1.10.3
- traefik

## To run
`sudo ./install.sh -u`
[Availble here](https://myweb.docker.localhost)

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
